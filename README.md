# Rafiester Portfolio - Modern Dynamic Web Application

A premium, interactive, and high-performance developer portfolio website built using **Vue 3** and **Vite**. The project features rich visual aesthetics, including a smooth WebGL interactive backdrop, custom pointer cursor tracking, a GSAP typing title effect, and a client-side LocalStorage Content Management System (CMS).

---

## 🚀 Key Features

* **Three.js WebGL Backdrop**: A GPU-accelerated interactive spotlight and fluid aurora simulation that dynamically responds to cursor movement.
* **Custom Cursor**: A smooth, customized cursor follower ring that tracks pointer coordinates locally within the hero viewport.
* **GSAP-powered Typing Title**: A typing text animation applied to the main Hero title, customizable via speed and cursor characters.
* **Local Storage CMS**: A fully functional admin dashboard allowing real-time edits to all layout copy and media sections without needing an active backend database.
* **Responsive Layout**: Tailored modern CSS layouts (supporting dark mode styles) designed to look stunning on both desktop and mobile devices.

---

## 🛠️ Technology Stack

* **Core**: Vue 3 (Composition API `<script setup>`), JavaScript
* **Build System**: Vite
* **3D & Shaders**: Three.js (WebGLRenderer, OrthographicCamera, custom GLSL Fragment Shaders)
* **Animation**: GSAP (GreenSock Animation Platform)
* **Icons**: Custom optimized inline vector SVGs
